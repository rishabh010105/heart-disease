Project Overview

This web application uses a trained k-Nearest Neighbours model to estimate the likelihood of heart disease based on vital signs, lifestyle factors, and lab results. It is built with Django on the backend and a clean, responsive UI suitable for demonstrations and viva.

Tech Stack: Django, scikit‑learn, pandas

Model Snapshot

The model has been trained on a cleaned patient dataset. Inputs are scaled and encoded before prediction to match the training pipeline, ensuring consistent results during deployment.
